INSTRUCTION MANUAL

Search for a Location:

  • Use the search bar at the top of the page to enter the name of the city or location you want to check the weather for.
  • Click on the "Search" button to fetch and display the weather information for the specified location.

Current Weather Overview:

  • Once you've searched for a location, the main section of the page will display the current weather information for that area.
  • Check the temperature, minimum and maximum temperature, humidity, cloud percentage, wind speed, wind degrees, sunrise, and sunset.

Popular Places Weather:

  • Explore the table showcasing the weather information for popular places such as Beijing, Hyderabad, Chennai, New York, London, and Adelaide.
  • View details like cloud percentage, feels like temperature, humidity, max and min temperatures, sunrise, sunset, temperature, wind degrees, and wind speed.

Navigation and Additional Information:

  • Navigate through the website using the navigation bar at the top, providing links to the home page, usage of the website, and the scalability and future of the websie.
  • Learn more about the website and its usage by clicking on the "About this website" link.

Responsive Design:

  • Enjoy a responsive design that adapts to different screen sizes, ensuring a seamless user experience on various devices, including desktops, tablets, and mobile phones.
  • The website is styled with a pleasing beige background and organized layout for easy readability and navigation.
